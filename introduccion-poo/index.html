Item 1

La propiedad Number.NEGATIVE_INFINITY representa el valor del infinito negativo. Es una propiedad estatica, por lo cual no hace falta instanciar un objeto de tipo Number para poder utilizarla.

Se podría utilizar la propiedad Number.NEGATIVE_INFINITY para indicar una condición de error que retorne un numero finito en caso de que esto suceda.

Dentro del proceso mencionado se utiliza para setear este valor a una variable para poder utilizarla para comparar contra los numeros almacenados en un array e identificar cual es el mayor de ellos.

Una alternativa es utilizar el valor -Number.MAX_VALUE, que es un valor finito muy pequeño que se utiliza para acercarse al infinito negativo.

El for of lo que hace es recorrer todos los elementos de un array uno por uno.

Los datos estan almacenados en la variable "nums" pasada por parametro.

La funcion recorre todos los datos del array enviado por parametro uno por uno, pero no hace nada con ellos ya que dentro de la condicion no hay nada definido. Para que tenga sentido, dentro de la condicion habría que agregar max_num = num, para que retorne el valor maximo encontrado, sino retorna "-Infinity" por defecto.
